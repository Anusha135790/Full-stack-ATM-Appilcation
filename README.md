## End to end project development steps

- step 1:
    Create an virtual environment <br>
    ```conda create -n flask_env python==3.13 -y```

- Step 2:
    Avtivate virtual environment <br>
    In cmd: ```conda activate flask_env``` <br>
    In gitbash: ```source activate flask_env```
- Step 3:
    Install pre-requirement modules<br>
    ```pip install -r requirements.txt```

- Step 4:
    Implement basic home route in app.py
